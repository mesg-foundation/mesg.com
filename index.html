What is MESG ?

MESG is an open-source decentralized infrastructure which makes all technologies on or off the blockchain interoperable. It’s a building platform for developers.

In this New Paradigm of application development, implementation of ideas is easy. We have taken down the barrier to developing dApps for a new generation of creatives and builders.

Why build with MESG ?

Developers and Businesses will experience different benefits by using MESG.

Tap into the infinite possibilities of combining existing services to execute solutions instead of being constrained by the complexities of each step in the process.

  • Technology Agnostic

    All Internet and blockchain technologies are compatible with MESG. Once a service is connected, it doesn’t need to be reconnected.

  • Decentralized

    The availability of decentralized processing allows you to pay only for what you use, and build applications without bottlenecks, overhead or a centralized point of failure.

  • Event Driven

    MESG’s architecture connects to all existing services and even the internet of things, to allow users to set up a series of tasks that will execute automatically once external events trigger it.

  • Easy to Develop

    MESG removes the steep barrier to implementing new technologies and services by removing the need to understand the context of technologies to benefit from their functionalities. Re-use the best solutions already created.

  • Efficient

    You have no boilerplate code to maintain. Simply focus on your business logic.

  • Adaptable

    Updating your application is now as simple as plugging in your new service.

Roadmap

  • May 1st 2018

    Core Launched

    Create your services and connect them together with your application through a single connection to Core, allowing Core to handle all communications and interoperability with any technology.

  • Q3 2018

    Rapid Deployment

    No need to code your application anymore, just send a list of events with corresponding tasks within a simple configuration file to Core which will then execute tasks on your application’s behalf.

  • Q4 2018

    Beta Network

    The decentralized beta Network means no coding or servers are necessary to run your applications.
    We will also launch an economy based on your participation in the Network.

  • Q3 2019

    Main Network

    MESG launches its own blockchain Network providing for full scalability and a cheaper and faster user experience.
